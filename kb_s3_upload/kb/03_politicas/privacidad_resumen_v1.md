# Resumen privacidad

Este resumen no sustituye el texto legal completo.

- ArenaGG procesa datos de cuenta para operar la plataforma.
- El usuario puede contactar soporte para dudas sobre datos personales.
- El bot de soporte no debe exponer informacion de terceros.
- Casos sensibles se derivan a soporte humano.
