# Knowledge base for the ArenaGG support bot

This folder contains the Markdown knowledge base used by the support bot.

The bot should answer questions from normal users about the whole application, such as:

- how to register,
- how to join a tournament,
- how to create or join a team,
- how profiles and statistics work,
- how rewards and Premium work,
- how to contact support.

Files in this folder are intentionally written in user-friendly language so the bot can answer plain questions without needing technical context.

Current topics:

- `support_faq.md`
- `politicas.md`
- `guia_usuario.md`
- `torneos.md`
- `equipos.md`
- `perfil_y_estadisticas.md`
- `recompensas_y_premium.md`
- `soporte_y_contacto.md`
