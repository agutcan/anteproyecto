# Perfil y estadisticas

## Datos de perfil

Un usuario puede editar:

- nombre,
- apellido,
- fecha de nacimiento,
- pais,
- biografia,
- avatar.

## Estadisticas visibles

El perfil muestra informacion como:

- partidas jugadas,
- partidas ganadas,
- winrate,
- MMR,
- renombre,
- monedas.

## Como se actualizan las estadisticas

Las estadisticas cambian cuando se juegan partidas oficiales y se registran resultados.

## Para que sirve el MMR

El MMR ayuda a ordenar los equipos y jugadores para emparejamientos mas equilibrados.

## Para que sirve el renombre

El renombre refleja reputacion o comportamiento dentro de la plataforma y puede subir o bajar segun la actividad.

## Ranking

El ranking permite comparar jugadores segun sus estadisticas publicas.
